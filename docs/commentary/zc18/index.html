I do not have much to say about this level.  I thought it would be fun to try to recreate the Portal level from Peggle Extreme in Portal since it had a physics engine and a ball that could bounce.  While it took a few iteratons to make the core peg-breaking experience feel good, it was honestly probably more work just wiring up all the indicator lights.  One of my favorite parts was the last added, which is if you run out of balls, you can try to use the missile turret to break the last remaining pegs (or you could let it hit you and start over).
